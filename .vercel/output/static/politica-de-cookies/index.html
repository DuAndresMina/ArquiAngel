4. Consentimientos

Almacenamiento publicitario

ad_storage

Permite almacenar datos relacionados con anuncios personalizados.

Datos de usuario

ad_user_data

Incluye información del usuario utilizada para personalizar anuncios.

Personalización

ad_personalization

Permite personalizar la experiencia del usuario en función de sus preferencias.

Análisis

analytics_storage

Se utiliza para recopilar datos sobre el uso del sitio web y generar estadísticas.

Funcionalidad

functionality_storage

Almacena configuraciones necesarias para el funcionamiento básico del sitio.

Seguridad

security_storage

Garantiza la seguridad del sitio y protege contra accesos no autorizados.
